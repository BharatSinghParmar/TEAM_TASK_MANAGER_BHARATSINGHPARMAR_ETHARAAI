# Team Task Manager

A role-based web application for managing projects, tasks, and team collaboration. Built with the MERN stack (MongoDB, Express, React, Node.js) and Tailwind CSS.

## 🚀 Quick Start

1. Install dependencies and build the frontend:
   ```bash
   npm run install-all
   npm run build
   ```

2. Start the production server:
   ```bash
   npm run start:prod
   ```
